== Description ==
1) Go to WordPress dashboard

2) Then settings->permalinks

3) Default permalink structure is look like
/%postname%/

4)Now remove all tag and add .html extension at the end of the permalink. shown below

EXAMPLE : /%postname%.html

Just Add .html


5)Now click on save changes button
Now your are done add html suffix in wordpress url


[Demo](https://bdcodemaker.com/demo/plugin/animcat/html-suffix.html)



== Installation ==

1) Upload 'html-suffix' to the '/wp-content/plugins/' directory

2) Activate the html-suffix plugin  

3) Go to settings->permalinks

4) Default permalink is look like /%postname%/

5)Now remove all tag and add .html extension at the end of the permalink. shown below

EXAMPLE : /%postname%.html

Just Add .html

6)Now click on save changes button
Now your are done add html suffix in wordpress url
